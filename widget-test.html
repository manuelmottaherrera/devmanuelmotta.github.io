Prueba del Widget de Chat

Esta pagina carga el widget de Tyse Helpdesk.

Para probar:

  1. Levantar la app: cd tyse-helpdesk && ./mvnw
  2. Exponer con ngrok: ngrok http 8086
  3. Reemplazar la URL de ngrok y la API key en este HTML
  4. Push a GitHub Pages y abrir esta pagina
La burbuja de chat debe aparecer en la esquina inferior derecha.
